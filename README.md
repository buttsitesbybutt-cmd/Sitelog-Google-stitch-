# 📖 Overview

**Al-injaz Management MEP** is a modern construction site management platform created specifically for:

- Site Foremen
- Site Supervisors
- Construction Managers
- Contractors
- Small Construction Companies

Instead of using paper records or scattered WhatsApp messages, the application provides one centralized digital workspace for managing multiple construction projects from any device.

The application works as both:

- 🌐 Progressive Web App
- 📱 Android APK
- ☁️ Cloud synchronized
- 📶 Offline capable

---

# 🎯 Problem It Solves

Managing multiple construction sites is difficult.

Typical problems include:

- Lost paper notes
- WhatsApp conversations scattered between workers
- No centralized project history
- Difficult progress tracking
- Manual report preparation
- Internet connectivity issues on construction sites

Al-injaz Management MEP solves these problems by providing one digital construction logbook with automatic cloud synchronization and offline support.

## 🏗️ Project Management

Each project contains:

- Project Title

- Villa Number
- Plot Number
- Start Date
- Category Assignment

- Project Status

Duplicate protection prevents the same Villa + Plot + Category combination from being created twice.
